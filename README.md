# nodejs_socket을 활용한 실시간 채팅



## 📄 Description

- 닉네임과 프로필 사진을 선택하여 실시간으로 채팅 할 수 있는 서비스입니다.
- 실시간 이용자 수를 확인 할 수 있고, 원하는 사용자에게 귓속말이 가능합니다.
- 이용자의 입장 및 퇴장, 닉네임과 프로필 사진을 변경할 때마다 메시지를 띄웁니다.
